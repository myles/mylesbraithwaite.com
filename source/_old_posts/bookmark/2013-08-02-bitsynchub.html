BitSyncHub is a service to automatically mirror Bitbucket Mercurial repositories on GitHub.

I am not really the biggest fan of Git (I only use it because if you want any contributions to you open source project people expect it to be on GitHub) so this mirror might be a great idea to migrate my code over to BitBucket.
